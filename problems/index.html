The Isogeny Problems


What particular goals will there be toward which the leading cryptographical spirits of coming generations will strive?

Lofty minds are led to strive for the advance of science by nothing more than by laying before them difficult and at the same time useful problems: this page enumerates the foremost unsolved problems in the domain of isogeny-based cryptography. The resolution of any of these profound questions would mark a monumental advance in our discipline, and the broader cryptographical sciences. Thus, we hear within us the perpetual call: There is the problem. Seek its solution. You can find it by pure reason, for in isogeny-based cryptography, as in mathematics, there is no ignorabimus.


Problem 1

The Supersingular Isogeny Problem

Given two supersingular elliptic curves \( E \) and \( E' \) over a finite field \( \mathbb{F}_{p^2} \), compute an explicit isogeny \( \varphi : E \to E' \).

A solution is accepted when either:

  • it gives a deterministic algorithm with time complexity below \( \mathcal{O} ( p^{\frac{1}{2}} ) \)
  • it gives a probabilistic algorithm with time complexity below \( \mathcal{O} ( p^{\frac{1}{2}} ) \) with non-negligible failure probablity
OR, a solution shows that such an algorithm cannot exist.

Reward: € 100

Problem 2

Vectorization for Oriented Elliptic Curves

Given two \( \mathcal{O} \)-oriented elliptic curves \( (E, \omega) \) and \( (E', \omega') \), with \( d = |\operatorname{disc}(\mathcal{O})| \), find an ideal \( \mathfrak{a} \) such that \( \mathfrak{a} \ast (E, \omega) = (E', \omega') \).

A solution is accepted when either:

  • it gives a deterministic algorithm with time complexity below \( L_d( 1/2 ) \),
  • it gives a probabilistic algorithm with time complexity below \( L_d( 1/2 ) \), with non-negligible failure probablity
OR, a solution shows that such an algorithm cannot exist.

Reward: € 100

Problem 3

Hashing into Supersingular Curves

Define a cryptographic hash function \( \mathcal{H} \) that takes a binary string \( \mathtt{m} \) and returns a supersingular \( j \)-invariant over \(\mathbb{F}_{p^2} \).

A correct solution contains the description of a hash function that satisfies the following properties:

  1. It is efficient and deterministic.
  2. It is hard to find a collision, that is, two binary strings \( \mathtt{m}_1 \) and \( \mathtt{m}_2 \) such that \( \mathcal{H}( \mathtt{m}_1 ) = \mathcal{H}( \mathtt{m}_2 ) \).
  3. It is hard to invert: given an \( h \) in the codomain, it is hard to compute a binary string \( \mathtt{m} \) such that \( \mathcal{H}( \mathtt{m} ) = h \).
  4. The \(j\)-invariants are uniformly distributed in the codomain.
OR, a solution shows that such a function cannot exist.

Reward: € 100

Problem 4

Optimal KLPT

Given a prime number \( p \), a maximal order \( O \) in \(B_{p, \infty}\), and a left integral \( O \)-ideal \( I \), find an equivalent ideal \( J \sim I \) of norm \( N(J) = \ell^e \).

A correct solution contains the description of an algorithm that satisfies the following properties:

  1. It is efficient and deterministic, or probabilistic with non-negligible failure probablity.
  2. The norm of \( J \) is (close-to) optimal.
OR, a solution shows that such an algorithm cannot exist.

Reward: € 100

Problem 5

Random Prime Isogenies

Given a curve \( E \) over a finite field \( \mathbb{F}_q \) and a positive integer \( n \), generate an isogeny \( \varphi : E \to E' \) of degree \( n \) where \( E' \) is some other curve.

A solution contains the description of an algorithm that satisfies the following properties:

  1. It efficiently computes the j-invariant of the codomain \( E' \).

  2. It efficiently computes \( \varphi(P) \) for any point \( P \in E(\mathbb{F}_q) \).
  3. It requires no a priori knowledge of the endomorphism ring \( \operatorname{End}(E) \).
OR, a solution shows that such an algorithm cannot exist.

Reward: € 100

Problem 6

Non-Smooth Interpolation

Given \( \operatorname{End}(E) \) and an isogeny \( \varphi : E \to E' \) in interpolation form, where the points may have large non-smooth order, find generators of the ideal \( I_\varphi \).

A solution is accepted when either:

  • it gives an efficient deterministic algorithm
  • it gives an efficient probabilistic algorithm, with non-negligible failure probablity
OR, a solution shows that such an algorithm cannot exist.

Reward: € 100

Problem 7

Strong Encryption Protocols

Construct post-quantum encryption schemes, whose security reduces to (a small variant of) the Endomorphism Ring Problem.

Several solutions can be accepted for the following constructions:

  1. An efficient post-quantum key exchange.
  2. An efficient post-quantum public-key encryption scheme.
  3. An efficient post-quantum identity-based encryption scheme.
  4. An efficient post-quantum attribute-based encryption scheme.
OR, a solution that shows that one of these cannot exist.

Reward: € 100

Problem 8

Solving The Quantum Gap

Give an efficient classical algorithm for any of the below problems, where we have efficient quantum algorithms.

  1. The recovery of \( \operatorname{End}(E') \), given an efficient representation of an isogeny \( E \to E' \), and knowledge of \( \operatorname{End}(E) \).
  2. The equivalence or inequivalence of vectorization and parallelization for isogeny group actions
OR, show that a classical reduction cannot exist.

Reward: € 100
